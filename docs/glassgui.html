Synopsis

glassgui [OPTIONS]

Description

glassgui(1) is a GUI front end for the glasscoder(1) audio encoder. It provides a convenient and discoverable means for creating and saving glasscoder(1) invocations as well as an instance management system for saving multiple sets of configurations on the local system.

Options

--autostart

Start encoding immediately upon startup.

--delete-instance=name

Delete the name instance from storage and then exit.

--instance-name=name

Load and save settings from/to the name instance.

--list-instances

Print the list of saved instances to standard output and then exit.
